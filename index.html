My Name is Justin Klein. I am currently pursuing a Bachelor of Science in Computer Science, expecting to graduate in May 2025. The website comprises a main page, an ecommerce section showcasing available items with cart functionality, and an orders page for cart inspection. The ecommerce page allows users to filter and sort items, enhancing the browsing experience. Meanwhile, the orders page provides flexibility, allowing users to specify date ranges and line limits for a detailed overview of closed shopping carts.


This ecommerce project is for CSE 383 for the Fall 2023 Semester.
